# ByteSampler 

The `byte_sampling` directory provides code that builds from the ByteSampler framework from the paper, ["Sampling from Your Language Model One Byte at a Time"](https://arxiv.org/abs/2506.14123) (Hayase et al., 2025). 
It includes custom code for anchored decoding at the byte level. 

Please check out the [official repo](https://github.com/SewoongLab/byte-sampler) for more details!
